# Dj Mood Front

> Application de génération de playlist correspondant a notre humeur

## Installation

1. Assurez-vous d'avoir Flutter installé
2. Clonez ce dépôt
3. Exécutez `flutter pub get`
4. Lancez avec `flutter run`

## Configuration
- Flutter 3.38.3
- Dart ^3.10.0 (ou version supérieure)

## Fonctionnalités prévues
- Gestion de playlists
- Effets sonores
- Analyse et choix d'humeur
